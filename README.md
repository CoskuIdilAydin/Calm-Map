# Calm-Map
Calm Map is a GIS application designed to connect folks to calming places in their community. This application was built over about 10 hours for the 2025 Girl Develop It Hackathon.

Instructions for contributing to the repo:
1. To update the repo, create a pull request to the dev branch and let Veronica know to approve (602)725-6970. Then she'll create a pull request to main.
2. Store images and any other misc docs in the 'docs' folder.
3. Store the GIS data in the data folder.
4. Store Leaflet code in the SRC folder.
